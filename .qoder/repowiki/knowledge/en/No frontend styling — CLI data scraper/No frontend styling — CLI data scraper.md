This repository contains a Node.js command-line script (`scrape_cfcpn.js`) that scrapes the CFCPN (金采网) procurement notice API and writes results to JSON and CSV files. There is no HTML, CSS, SCSS, Tailwind, theme configuration, or any other frontend styling code in the project. The `frontend_style` category does not apply.
